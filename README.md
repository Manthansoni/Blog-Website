# Blog-Website

### `npm start client`

Runs the client app.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### `npm start server`

Runs the server.\
Open [http://localhost:8000](http://localhost:8000) to view it in your browser.

### `npm start dev`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
